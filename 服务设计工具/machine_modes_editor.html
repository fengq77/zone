提供的端口连接编辑器提供以下列：

Machine Modes Editor

Machine Deployment：将机器部署显示为容器机器模式和服务发现配置。如果没有机器列中存在部署，显示组件部署包。通过创建 Machine Deployment 表格单元格中的“+”按钮。

Machine：显示代表 Machine 的 Process Units。通过下拉列表或拖拽选择。

Function Group：显示Mode Declaration Group Elements。功能组捆绑进程，从而启用和终止进程组。一个功能组可能有许多功能组状态。如果一个一个进程的启动依赖指向一个功能组状态，然后启动过程取决于此状态。通过“+”按钮创建。

Mode Declaration Group：显示 Mode Declaration Group元素。通过下拉列表选择。

Mode值：默认，亦可支持新建、修改、设置默认值
