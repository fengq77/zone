数据类型包结构

1、类型包新建数据类型：

      Base Package：默认加载基础数据类型

      *基础数据类型有：bool、uint8、uint16、uint32、uint64、double、float、string、int8、int16、int32、int64

      Array Package：FixArray和DynamicAarry类型数据

      Struct Package：Stuck数据类型

      Union Package：Union数据类型

2、点击对应的数据类型包后，列表显示该类型下所有的数据



3、数据类型name不能重名（全局）

1、点击对应的数据类型Package，右侧显示列表数据，Property View显示默认空图片

2、点击列表数据，Property View显示该数据的详情及其子数据

3、另base datatypes 无 子数据，则不显示sub element info列表

数据类型列表

1、点击对应的数据类型package，列表显示对应的数据列表

2、点击单条数据，可在【Property View】查看其详细信息（含子元素）

3、窗口名称，根据不同package显示不同名称，如

     Base DataTypes List

     Array DataTypes List

     Struct DataTypes List

     Union DataTypes List

列表数据操作

1、【新增】可新增一条数据类型

2、【删除】选中的数据删除，点击删除后，需判断是否被引用，

是：toast提示“该数据类型被引用，删除失败！”

否：弹框提示是否删除该数据类型

复杂数据类型-设置属性值（基础数据类型无设置项）

页面交互
