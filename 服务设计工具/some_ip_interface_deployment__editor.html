SOME/IP 接口部署编辑器提供以下列：

Event Group说明：

Service Interface：所选服务中服务接口名称。

SOME/IP Interface Deployment： SOME/IP 服务接口为服务接口创建的部署。

Interface ID：SOME/IP 服务接口部署的 ID。意为中间件和总线上的接口，并从服务定义。可在此处编辑 ID。

Major Version：SOME/IP 服务接口的主要版本号部署。

Minor Version：SOME/IP 服务接口的次要版本号部署。

Deployment Element：部署元素服务接口。

Deployment Sub-Element：服务接口的子元素的部署元素

Tranport Protocol：传输协议。通过下拉列表选择（下拉选项“UDP”“TCP”）；

Method/Event ID：SOME/IP的方法/事件 ID。

Event Group：事件组。通过创建事件组单元格中的按钮“+”。
