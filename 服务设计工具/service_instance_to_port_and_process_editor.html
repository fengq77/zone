SOME/IP 服务实例编辑器提供以下列：

Service Instance

Adaptive SW Component

Process

> Service Instance：显示所选组件中包含的硬件组件包裹。

> Service Instance Mapping：显示映射到硬件组件。

> Software Port：显示软件组件的端口。

> Adaptive SW Component：显示 SOME/IP 服务接口实现分配给端口的服务接口。

> SOME/IP Service Instance：显示对应的 SOME/IP 服务实例。

> Process：输入提供的 SOME/IP 服务的服务实例 ID实例。在服务发现阶段，这个唯一的 ID 帮助客户端找到兼容的服务器。
