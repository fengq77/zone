> Unicast

> IPv6 Address：通过创建 IPv6 地址

 表格单元格中的按钮。可以使用 IPv4 或 IPv6，混合协议是不允许。

> UDP Port：通过下拉列表选择 Udp Tps。如果同时使用 UDP 和 TCP 端口已定义，在 SOME/IP 接口部署部分中进行的设置为考虑决定哪些内容由哪个端口发送。
