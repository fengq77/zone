File Storage列表说明

1、Instance Name：Instance的名称，本page页不重复

2、URI：落盘的完整路径

3、Default URI：默认值文件的存储完整路径

4、Access Mode：配置Instance读写权限，下拉单选，选项为【可读可写】【只读】【只写】

5、Backup Rate：输入数字；范围大于0的正整数；

                          鼠标移入「？」显示提示为

                         “设置备份文件的频率，如：

                            0：不备份

                            1：每次SyncToFile都备份

                            2：每2次SyncToFile备份1次

                            3：每3次SyncToFile备份1次”

6、操作项：

修改：修改对应的数据，修改后，支持“保存”“取消”

具体UIUE见UI交互图

Key Value Storage列表说明

1、Instance Name：Instance的名称，本page页不重复

2、URI：落盘的完整路径

3、Default File：默认值文件的存储完整路径

4、Access Mode：配置Instance读写权限，下拉单选，选项为【可读可写】【只读】【只写】

5、File Mode：文件类型；下拉单选；选项为【数据库】【文本文件】

5、Backup Rate：输入数字；范围大于0的正整数；

                          鼠标移入「？」显示提示为

                         “设置备份文件的频率，如：

                            0：不备份

                            1：每次SyncToFile都备份

                            2：每2次SyncToFile备份1次

                            3：每3次SyncToFile备份1次”

6、操作项：

修改：修改对应的数据，修改后，支持“保存”“取消”

具体UIUE见UI交互图

新增、删除

1、新增：新增一行

2、删除：点击“删除”后，出现多选框，可删除多条数据
