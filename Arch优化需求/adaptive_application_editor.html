Adaptive Application：显示所选应用程序中包含的自适应应用程序软件包。通过创建自适应应用程序表格单元格中的        按钮或表格工具栏中的       按钮。

Version：输入自适应应用程序的版本号。

Executable：显示自适应应用程序的可执行文件。创建可执行文件通过表格单元格中的       按钮。

Build Type：通过下拉列表为每个可执行文件选择构建类型。选择项“Debug”“Releace”

Version：输入可执行文件的版本号。

Minimum Time Granularity：输入最小时间粒度可执行文件。

Software Component Type：选择自适应应用软件组件类型(Adaptive Application SW Component Type)，由可执行文件通过下拉列表执行。



如果将 DaVinci Developer Adaptive 用作代码生成工具，还有一个必须设置属性才能完成自适应应用程序的配置：

1. 进入自适应应用程序的 General 属性页。

2. 通过下拉列表选择 Adaptive Application Kind。

该属性对应于 AUTOSAR 中的 Executable.category 属性和分配的可执行文件的种类
