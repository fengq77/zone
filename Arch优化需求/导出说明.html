PREEvision 可以导出 AUTOSAR 标准中描述的所有自适应清单。

服务接口描述包含一个或多个服务接口的详细信息，包括数据类型、数据类型映射、计算方法和单位。该文件可用于将服务接口定义从 PREEvision 传输到另一个工具，例如 DaVinci Developer Adaptive、CANoe 或其他安装的预视。 它还可以作为自适应设计的起点。

创建和导出服务接口描述：

1. 选择一个或多个服务接口。 注意：导出支持多选。

2. 打开上下文菜单并选择Export | AUTOSAR Service Interface Description。

> Service interface description（服务接口说明）

执行清单提供了实际部署所需的信息 ，即平台上的应用程序。

要创建和导出执行清单：

1. 在软件部署下，选择一个或多个进程。

2. 打开上下文菜单并选择导出 | AUTOSAR 执行清单。

> Execution manifest（执行清单）

可以导出以下描述和清单：

> Service interface description（服务接口说明）

> Execution manifest（执行清单）

> Machine manifest（机器清单）

> Service instance manifest（服务实例清单）

> Application description（应用说明）

机器清单为一个部署提供了特定于机器的配置

过程，独立于任何服务实例或应用程序。

要创建和导出执行清单：

创建和导出机器清单：

1. 在硬件组件包中的一个或多个 ECU 下，选择一个或多个处理单元。 注意：导出支持多选。

2. 打开上下文菜单并选择导出 | AUTOSAR 机器清单。

> Machine manifest（机器清单）

服务实例清单包含面向服务的配置详细信息考虑到底层传输协议的通信。 详细：它包含服务实例、服务发现设置、服务实例到的映射机器和可选的服务实例到软件组件的映射港口。 此格式在部署时使用。

要创建和导出服务实例清单：

1.在一个以太网通信集群下，选择一个或多个Consumed SOME/IP服务实例或提供的某些/IP 服务实例。 注：支持多选。

2. 打开上下文菜单并选择导出 | AUTOSAR 服务实例清单。

> Service instance manifest（服务实例清单）

创建了自适应应用程序描述。 它包括所有AUTOSAR 定义的清单内容在一个文件中。 它可以用于部署一个自适应应用程序或将数据传输到其他工具，例如，DaVinci Developer Adaptive 或 CANoe。

要创建应用程序描述，例如：

1. 选择一个或多个自适应应用程序。 注意：导出支持多选。

2. 打开上下文菜单并选择导出 | AUTOSAR 应用说明。

> Application description（应用说明）
